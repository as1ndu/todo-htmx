# todo-htmx

Todo List With HTMX

[View Demo](https://todo-htmx.pages.dev/)

#### Running the project
Clone the repo, install [wrangler](https://developers.cloudflare.com/workers/wrangler/install-and-update/) `npm install wrangler --save-dev`

Then run the following.

```bash
$ cd todo-htmx
$ chmod +x run.sh
$ sudo ./run.sh
```
